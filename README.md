## 前言

欢迎来到我们的汽车保养系统项目，这是一个基于SSM框架和微信小程序的汽车保养服务系统。该项目致力于为用户提供便捷、高效的汽车保养服务，通过互联网技术，实现线上预约、线下服务的模式。以下是项目相关内容的详细介绍。

## 内容介绍

汽车保养系统主要包括以下功能模块：用户模块、预约模块、服务模块、订单模块和管理模块。用户可以通过微信小程序查看附近汽车保养门店的信息，预约保养服务，并实时跟踪订单状态。同时，系统还为管理员提供门店管理、订单管理等功能，提高工作效率。

## 技术介绍

- 语言：Java
- 使用框架：Spring、Spring MVC、MyBatis、微信小程序
- 前端技术：JS、Vue、CSS3、Uniapp
- 开发工具：IDEA/Eclipse、Uniapp
- 数据库：MySQL 5.7/8.0
- 数据库管理工具：phpstudy/Navicat
- JDK版本：jdk1.8
- Maven：apache-maven 3.8.1-bin
- 前端环境：Node.js 12\14\16
